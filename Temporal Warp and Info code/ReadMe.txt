This folder contains MATLAB code used for trial-structured and event-aligned analyses described
in "The Primate Hippocampus Constructs a Temporal Scaffold Anchored to Behavioral Events".


Primary analysis scripts:
- ymazemodule_EVTnew.m
- ymazemodule_warp2.m
- evt_remappingprocessing2.m
- ymazemodule_EVTnew_TT.m

These functions orchestrate event-aligned trial warping, template
construction, and information-based analyses.

Supporting / utility functions:
- info_MC2.m
- infoEVT_MC3.m
- infocalc.m
- Olypherinfo.m
- reliablecorr2.m
- TimeTemplateSetUp.m
- warptrial_fulltrial4info.m
- mergestruct.m
- evt_remappingtest2.m

Some scripts also contain historical variants or extensions developed
for adaptations of analyses in the paper.
